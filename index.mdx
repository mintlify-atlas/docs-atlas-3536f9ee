## Overview

NestJS Better Auth is a powerful integration library that brings the full capabilities of [Better Auth](https://www.better-auth.com/) to your NestJS applications. It provides a complete authentication and authorization solution with TypeScript-first design, automatic route protection, and seamless integration with NestJS's dependency injection system.

## Quick Example

Here's how simple it is to add authentication to your NestJS application:

```typescript main.ts
import { NestFactory } from "@nestjs/core";
import { AppModule } from "./app.module";

async function bootstrap() {
  const app = await NestFactory.create(AppModule, {
    bodyParser: false, // Required for Better Auth
  });
  await app.listen(3000);
}
bootstrap();
```

```typescript app.module.ts
import { Module } from "@nestjs/common";
import { AuthModule } from "@thallesp/nestjs-better-auth";
import { auth } from "./auth";

@Module({
  imports: [
    AuthModule.forRoot({
      auth,
      bodyParser: {
        json: { limit: "2mb" },
        urlencoded: { limit: "2mb", extended: true },
      },
    }),
  ],
})
export class AppModule {}
```

```typescript user.controller.ts
import { Controller, Get } from "@nestjs/common";
import { Session, UserSession } from "@thallesp/nestjs-better-auth";

@Controller("users")
export class UserController {
  @Get("me")
  async getProfile(@Session() session: UserSession) {
    return session.user;
  }
}
```

All routes are automatically protected. Use decorators to allow public access or add role-based restrictions.
